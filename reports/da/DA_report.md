# Отчёт по Data Analysis (GA / СберАвто)

**Дата:** 14 Feb 2026  
**Цель:** проверить продуктовые гипотезы про конверсию в целевые действия и ответить на вопросы продуктовой команды.

---

## 1) Что считаем целевым действием (Target)

Целевое действие = любой hit с `event_action` из списка:

- `sub_car_claim_click`
- `sub_car_claim_submit_click`
- `sub_open_dialog_click`
- `sub_custom_question_submit_click`
- `sub_call_number_click`
- `sub_callback_submit_click`
- `sub_submit_success`
- `sub_car_request_submit_click`

Дальше **CR (Conversion Rate)** считаем как:

> **CR = (кол-во сессий с хотя бы 1 целевым событием) / (все сессии)**

---

## 2) Срезы и определения

- **Organic / Paid:** классификация по источнику/каналу (как в ваших агрегатах).
- **Mobile / Desktop:** тип устройства (как в ваших агрегатах).
- **Города присутствия:** *Москва и область* + *Санкт‑Петербург* (presence), остальное — other.

### 4.3. Стоит ли увеличивать присутствие в соцсетях?

В текущей выгрузке `utm_source`/`utm_campaign` выглядят **обфусцированными (захэшированными)**, поэтому “соцсети” напрямую по названию источника/кампании выделить нельзя.

Что можно сделать корректно:
1) Если в данных есть `utm_medium` (например, `social`, `cpc`, `banner`, `cpm`) — сделать срез по нему и сравнить **volume + CR**.  
2) Если у вас есть справочник соответствий (что зашифровано под каждым `utm_source`/`utm_campaign`) — добавим “соцсети” как отдельную группу и повторим анализ.

Пока рекомендация по текущим данным:
- **увеличивать** стоит те каналы/кампании, которые одновременно:
  - дают **существенный объём**, и
  - имеют **CR выше среднего** (или статистически значимо выше), и
  - имеют стабильный 95% доверительный интервал (не слишком широкий).
- каналы с высоким CR, но малым объёмом — сначала “проверять” (масштабировать осторожно).

## 5) Резюме для бизнеса (1 экран)

- По всем трём гипотезам получились **статистически значимые различия по CR** между сравниваемыми сегментами.
- Для перераспределения усилий ориентируемся на **компромисс “объём × CR”**:
  - масштаб — ТОП источники/кампании по объёму;
  - эффективность — ТОП источники/кампании по CR при достаточном объёме.
- По авто:
  - есть бренды/модели, которые дают и высокий спрос, и высокий CR — кандидаты для приоритизации в выдаче/контенте/промо.
